## 1. Keyword strategy

Target a ladder from high-intent niche to broad head terms. Niche terms convert and rank faster early; head terms are long-term goals.

Primary (core intent): memory game, brain game, memory training, concentration, focus.
Secondary (mechanic/feel): pattern memory, sequence memory, relaxing puzzle, calm game, mindful, zen.
Long-tail (fast wins): daily brain puzzle, memory game offline, calm memory game, brain training daily, pattern recall game.
Brand: glowbloom, glow bloom.

Rules: never repeat a word already used in the app name/title (it's already indexed). On Apple, no spaces after commas in the keyword field (saves characters); singular form usually covers plural. Don't keyword-stuff the Play description — write naturally; Google indexes the whole thing and penalises spam.
